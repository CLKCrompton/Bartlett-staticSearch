Test data set

Documents in the collection: badthings.html, badthings.html, clues.html, ditty.html, shizukesay.html, twighlight.html, booze.html, decade_year.html, and of course search.html.

This is the index page for the Static Search test data set. This data set is designed to cover a range of different input document types and configurations in order to test a variety of scenarios the codebase needs to handle, including:

  • Nested folders.
  • Documents with and without ids (this has one).
  • With and without titles (this has one).
  • Good range of HTML elements providing scope for testing context stuff.
  • Foreign languages and non-latin stuff such as Japanese.
  • Hyphenated terms, single and double, such as one-way street and hard-to-process sequences.
  • Names, both in sentence-initial and sentence medial positions. Fred Bloggs and Mary Wollstonecraft Shelley are examples.
  • Numbers such as 25, 2,302, and 3.14159, or currency amounts such as £4,500 or ¥9,000,000.
  • Items that should not be indexed (such as line numbers attached to poetic lines).
  • Constrained contexts which should delimit keyword-in-context ranges, such as table cells.
  • Inline elements whose boundaries should be ignored when indexing.
